## Sobre

Este é um site protótipo que será usado como inspiração para a equipe de desenvolvimento da empresa do cliente.

## Tecnologias utilizadas

- Next JS
- Typescript
- Tailwind
- Framer-Motion
- React-hook-form
- Zod
- Vercel (deploy)
